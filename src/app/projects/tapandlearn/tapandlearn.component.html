Tap and Learn Languages

Project description

After finishing the Web Engieering MOOC course of the Technische Hochshule Mittelhessen I deveoped my first (and last) Sencha Touch application. The product was a vocabulary improoving game. The development was supervised a language school. Here, there can be found an interview with the details.

The application

The application was a game developed in 3 different languages. The users could choose a topic from a list and start the game. The users could see an image, and 4 letter. The aim was to choose the letter by which the word stated shown on the picture. If the chice was good the applicationloeded a new image, if no, the player could choose whethet want to the right answer or just jump to the next one. The application calculated the performance, and stored, hence the user could control his/her improvment.
